# Python Fetching Data and Manipulation Challenge

## Concepts Used:
1. Importing `requests` module to send requests
2. Importing `json` module to handle JSON Decode Errors
3. Using `for` loops to loop over fetched data and append to html
4. Use `.replace()` to append `html` elements into `<title>` `<head>` and `<body>` elements
5. Use `try/except` statements to control flow of errors and exceptions
6. Use `tailwindcss` to style the tables
